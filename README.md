# emojxbot

## Requirements
- Python 3.11

## Installation
### Clone repository
```bash
$ git clone
$ cd emojxbot
```

### Setup virtual environment and install dependencies
```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

### Configure `.env` file

## Usage
### Start long polling:
```bash
$ cd weather-bot/bot
$ source .venv/bin/activate
$ python3 bot_polling.py
```
